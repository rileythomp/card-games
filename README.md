# Card Games

https://cardsfun.herokuapp.com

A card games site including Concentration and War.

Made with Node.js and vanilla JS.

To run locally:

```$ git clone https://github.com/rileythomp/card-games.git```

```$ npm i```

```$ npm start```

Then visit localhost:3032
